terrain-gen - Procedural Terrain Generation in C++
This was originally a final project for my graphics class. Terrain is generated using Perlin noise and then the surface of said noise function is approximated using the Marching Cubes algorithm. All of this is done in the C++ code, so it does run a little slowly, but the results are pretty.
