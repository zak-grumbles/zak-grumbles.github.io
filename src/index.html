Projects
terrain-gen - Procedural Terrain
This was originally a final project for my graphics class. Terrain is generated using Perlin noise and then the surface of said noise function is approximated using the Marching Cubes algorithm. All of this is done on the CPU so it does run a little slowly, but the results are pretty. Source code can be found here.
Dank Dungeoneering
A dank platforming game where you craft your own dungeon while conquering others. Developed in 3 months for university capstone project, the game was created using Unity and optimized for the Nvidia Shield tablet. I was responsible for the server backend , the UI, and the enemy AI. The server code can be found here. Unfortunately, due to licensing whatnot, I can't distribute the game or its source.
Dare2Dream Engine
What an awful title, right? This is a very rudimentary 2D game engine that I developed with two other classmates for a Gaming Platform Frameworks class. It featured basic 2D physics, saving and loading of scenes, a GUI editor, and audio. Source code is available here, although I must warn you; It's painfully obvious that it was written by undergrads. It was written specifically for the university machines, so I make no promises about whether or not it'll work on any given machine or other Visual Studio versions.
